1. Prompt & Router Decision
Example Prompt:
"How are smart watches selling in the last 7 days compared to the same week last year?"
ADK Router Reasoning
• Transactional & Realtime: Routes to AlloyDB for recent transactional metrics.
• Historical & Governed: Queries Looker semantic layer for governed benchmarks.
• Analytical: Accesses BigQuery for unstructured sentiment analysis.
2. Specialist Execution
Real-time 💿 AlloyDB Agent 
SELECT count(*) FROM orders 
WHERE category = 'smartwatch' 
AND created_at >= current_date - 7
Returns: 4,258 orders
Historical 👁️ Looker Agent 
Explore: gadgets_transactions
Filters: smartwatch, week_of_year=15, 2025
Returns: 3,890 benchmarks
Analytics 📊 BigQuery Agent 
SELECT sentiment, comment 
FROM cymbal_gadgets.reviews
Returns: 120 reviews
3. Final Response Synthesis
ADK Aggregator

Synthesizes transactional databases with aggregate explorers synchronously.

Markdown Output:

Smart watch sales are robust! We recorded 4,258 transactions in the last 7 days. This represents a 9.4% increase compared to the 3,890 benchmarked sales of the same week last year.
